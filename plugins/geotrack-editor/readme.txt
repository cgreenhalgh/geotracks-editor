== Description ==
Plugin for wordpress which allows authoring of geotracks and geolists, i.e. geo-located playlists.

== Installation ==
1. Upload the directory `geotrack-editor` to the `/wp-content/plugins/` directory
1. Activate the plugin through the \'Plugins\' menu in WordPress
